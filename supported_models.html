Supported Models#

This page contains table of pre-trained models with information about model support on various inference backends and CPU/GPU devices.

Each model has link (under model name) to original documentation with download instructions.

Most models are from OpenVINO™ Open Model Zoo but some models are from other sources (see column Source Repo).

Abbreviations used in the table#

Abbreviation

Description

openvino

OpenVINO™ toolkit - as model file format (*.xml + *.bin) and inference backend in Intel® DL Streamer

pytorch

PyTorch* framework - as model file format and inference backend in Intel® DL Streamer

tf

TensorFlow* framework - as model file format and inference backend in Intel® DL Streamer

onnx

ONNX - Open Neural Network Exchange file format

caffe

Caffe* framework - as model file format

dlstreamer

Intel® DL Streamer
